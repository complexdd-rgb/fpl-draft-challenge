# FPL Daily Challenge Ideas

## Prompt system

- Give every prompt a unique ID.
- Store its position, difficulty, tags and rating.
- Track the last date it was used.
- Automatically block prompts used in the previous seven days.
- Show how many valid players match each prompt.
- Warn when one player is clearly the best answer.
- Mark prompts as active, resting or retired.

## Daily challenge generator

- Select one goalkeeper, four defenders, four midfielders and two forwards.
- Require at least three anti-meta prompts.
- Prevent duplicate conditions.
- Estimate challenge difficulty.
- Preview valid answer counts.
- Export the chosen prompts directly into the JavaScript challenge block.
