# FPL Daily Challenge Prompt Library

This is the master prompt bank for the FPL Daily Challenge game.

## Daily selection rules

1. Select one goalkeeper prompt.
2. Select four defender prompts.
3. Select four midfielder prompts.
4. Select two forward prompts.
5. Include at least three anti-meta prompts.
6. Avoid prompts used in the previous seven challenges.
7. Check that every prompt has several valid answers.
8. Rewrite prompts whose obvious answer is always Salah, Haaland, Kane, Palmer or Alexander-Arnold.

Ratings:

- ⭐⭐⭐⭐⭐ Excellent
- ⭐⭐⭐⭐ Strong
- ⭐⭐⭐ Useful
- ⭐⭐ Occasional
- ⭐ Retire or rewrite

Avoid these unless combined with a strong anti-meta modifier:

- Player with 20+ goals
- Player with 15+ assists
- Defender with the most clean sheets
- Highest-scoring player in a position
