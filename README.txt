FPL Daily Challenge 4.0

Upload all four files to the same GitHub repository folder:

- index.html
- daily-challenge.js
- visuals.css
- enhancements.js

The updated index.html retains the embedded database and built-in challenges as a fallback.
The newest external challenge is loaded after those fallbacks and is therefore selected by
the existing 'last challenge' loader.

For future daily updates, normally replace only daily-challenge.js.
